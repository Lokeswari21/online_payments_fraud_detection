# Online Payments Fraud Detection

This project predicts whether an online payment transaction is fraudulent or legitimate using machine learning techniques.

## Technologies Used
- Python
- Flask
- Machine Learning (Random Forest)
- HTML, CSS
- GitHub

## Features
- User-friendly web interface
- Fraud prediction using trained ML model
- Flask-based deployment

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Train the model  
   `python training/train_model.py`

3. Run Flask app  
   `python flask/app.py`

4. Open browser  
   `http://127.0.0.1:5000/`
